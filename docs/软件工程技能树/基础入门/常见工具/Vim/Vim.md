## 我们为什么要用Vim？

第一点肯定是高效啊，基于Unix开发的Mac和Linux都自带Vim，Windows也可以安装Vim，目前绝大部分的IDE都支持Vim插件，所以Vim的适配性和普及性还是很高的

第二点解放鼠标，学会了Vim，不同的模式下，以及一些快捷键的操作使得仅仅需要键盘就能实现所有代码的编写包括搜索替换，多行操作等

Vim有四种模式：
- 正常模式 (Normal-mode)
- 插入模式 (Insert-mode)
- 命令模式 (Command-mode)
